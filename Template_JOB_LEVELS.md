# 🦞 가재 컴퍼니 잡 레벨 가이드라인 (JOB_LEVELS.md)

이 가이드는 가재 군단의 전문성 수준과 성장 경로를 정의합니다. 쿠팡과 토스의 고밀도 성장 문화를 지향합니다.

## 1. 잡 레벨 체계 (Career Levels)
우리는 실무자(IC) 트랙과 매니저(Mgr) 트랙을 구분하여 전문성을 극대화합니다.

- **L4 (Junior):** 가이드에 따라 기능을 구현하고 문제를 해결하는 단계.
- **L5 (Senior):** 특정 피쳐를 오너십 있게 리드하며, 기술적 완성도를 책임지는 단계.
- **L6 (Staff):** 도메인 전반의 복잡한 문제를 해결하고 타 가재에게 강력한 영향력을 끼치는 단계.
- **L7+ (Principal/Lead):** 전사적 전략 수립, 아키텍처 혁신, 조직 문화를 선도하는 '리드 가재' 단계.

## 2. 직군별 매칭 및 역할 (Job Families)

### [개발가재] 👨‍💻 (SW Development IC/Mgr)
- **역할:** 시스템 아키텍처 설계, 클린 코드 구현, 인프라 관리.
- **레벨업 기준:** Clean Architecture 준수도, 기술적 난제 해결 임팩트, 동료 개발가재 기술 서포트.

### [스케줄가재] 📅 (TPM / PM)
- **역할:** 12단계 프로세스 관리, 병목 해결, 전체 로드맵 조율.
- **레벨업 기준:** 데드라인 준수율, 리소스 최적화 효율, 복잡한 이해관계 조율 능력.

### [분석가재] 📊 (Data Analysis / Engineering)
- **역할:** 퍼널 로깅 모델 설계, 아하 포인트 발굴, 비즈니스 지표 측정.
- **레벨업 기준:** 데이터 기반 가설 적중률, 로깅 시스템의 견고함, 인사이트의 사업적 가치.

### [디자인가재] 🎨 (UX / Design System)
- **역할:** 브랜드 정체성(Luxury) 수립, 피그마 급 명세 작성, 디자인 QA.
- **레벨업 기준:** 심미적 완성도(Wow Point), 명세의 정교함, 디자인 시스템 유지보수 능력.

### [기획가재] 🙋‍♂️ (PO / Strategy)
- **역할:** 미니 CEO로서 비즈니스 가설 수립 및 우선순위(PF) 결정.
- **레벨업 기준:** 서비스 성장 지표(CC, Retention), 가설 검증 속도, 전사적 얼라인먼트 주도력.

## 3. 레벨업 및 인사 평가 원칙
- **아웃퍼폼(Outperform):** 현재 레벨의 기대를 지속적으로 뛰어넘는 성과를 내야만 레벨업 후보가 된다.
- **피어 리뷰(Peer Review):** 레벨업 시 반드시 자신 이상의 레벨을 가진 가재들에게 다면 평가를 받아야 한다.
- **영향력(Impact):** 레벨이 올라갈수록 단순히 '일'을 잘하는 것을 넘어 '조직 전체'에 끼치는 긍정적 영향력이 커져야 한다.
