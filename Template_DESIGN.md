# Design System & UI Specification

## 1. Visual Direction: "Medical Trust"
- **Color**: White base with Medical Blue (#3B82F6) accents.
- **Typography**: Noto Serif KR (Headers), Pretendard (UI/Body).
- **Vibe**: Clean, Professional, Safe, Emotional.

## 2. Technical Stack
- **Styling**: Tailwind CSS V4.
- **Interactions**: Framer Motion (Smooth entrance and scroll parallax).

## 3. Key Layouts
- **Landing**: Vertical scroll-based storytelling.
- **B/A Slider**: Interactive comparison tool.
- **Result Grid**: Premium 3-card layout for the generated photos.

## 4. Design-to-Dev Handover
- All assets optimized in WebP format.
- Mobile-first responsive design (Breakpoint: 768px).
