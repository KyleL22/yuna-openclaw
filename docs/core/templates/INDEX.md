# 🗂️ 도메인 인덱스 (Domain Index): [CORE/TEMPLATES]

## 1. 개요 (Overview)
본 문서는 가재 컴퍼니의 모든 지능 산출물을 찍어내는 **'성역의 금형(Sacred Templates)'** 리포지토리입니다. 가재 군단의 모든 기록은 아래 정의된 규격에 따라 생성되어야 하며, 규격 미준수 시 지능의 가치로 인정받지 못합니다.

## 3. 집행 규율 (Enforcement)
1. **[P0] 규격 엄수**: 모든 신규 문서 생성 시 위 템플릿을 복제하여 사용하며, 구조의 임의 변경을 금지한다.
2. **[P0] 매핑 준수**: 생성된 문서는 반드시 `TEMPLATE_MAPPING.md`에 정의된 공식 경로에 위치해야 한다.
3. **[P1] 템플릿 업데이트**: 템플릿 자체의 수정은 반드시 '가재 평의회'의 합의와 CEO 승인을 거쳐야 한다.
