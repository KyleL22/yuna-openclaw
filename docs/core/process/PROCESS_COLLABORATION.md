# 🦞 가재 컴퍼니 전사 협업 프로토콜 (PROCESS_COLLABORATION.md) v1.5

본 문서는 모든 가재가 공통적으로 준수해야 할 협업의 대원칙을 정의합니다. (상세 보고 형식은 헌법 제 2 조를 따른다.)

---

## 1. 디자인-개발 싱크 (Sync)
- **명세의 정교화:** UX가재는 레이아웃, 스페이싱, 애니메이션 등을 텍스트로 치밀하게 묘사하여 전달한다.
- **적극적 질의:** 개발가재는 구현 중 의문 발생 시 즉시 UX가재에게 질문하여 기획 의도를 확인한다.

## 2. 갈등 중재 루프
- 가재 간의 정책 충돌이나 의사결정 지체 발생 시, 해당 프로젝트의 책임 노드(Value Arbitrator 등)를 소집하여 최종 판결을 요청한다. (헌법 제 6 조 2항)

---
**지휘 지침:** "지능은 고립될 때 썩고, 충돌하며 리뷰할 때 비로소 완성된다." ⚔️🚀
